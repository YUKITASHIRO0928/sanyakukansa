  散薬調剤支援システム — 店舗セットアップ手順
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【必要なもの】
  ✅ Windows PC（ELIXIRと同じネットワーク）
  ✅ Node.js（https://nodejs.org/ からLTS版）
  ✅ インターネット接続

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【初回セットアップ（店舗ごとに1回だけ）】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  手順1: このフォルダをPCに置く
          （GitHubからZIPダウンロード → 展開）

  手順2: setup-store.bat をダブルクリック

  手順3: 画面の指示に従って入力する
          ・店舗名（例: ○○薬局 北店）
          ・ELIXIRサーバー名（通常: ELIXIR1）
          ・ポート番号（通常: 3456）

  手順4: 自動起動のセットアップも続けて実行

  手順5: open-firewall.bat を右クリック →「管理者として実行」
          （他のタブレット・PCからアクセスする場合のみ）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【毎日の使い方】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  PC起動 → 自動でサーバーが起動
  ブラウザで http://localhost:3456 を開く

  ※ タブレット・他PCから使う場合:
     サーバーPC起動後、黒い画面に表示されるIPアドレスを使用
     例: http://192.168.1.10:3456

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【トラブル対応】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ・画面が開かない → ブラウザで http://localhost:3456 を開く
  ・サーバーが落ちた → pm2 resurrect（自動で復旧するはず）
  ・ログ確認 → pm2 logs sanyakukansa
  ・手動再起動 → pm2 restart sanyakukansa

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【複数店舗に展開する場合】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  各店舗のPCで setup-store.bat を実行するだけ。
  店舗ごとに config.json が作成されます。
  アプリの更新は GitHub に push するだけで全店舗に自動反映。
